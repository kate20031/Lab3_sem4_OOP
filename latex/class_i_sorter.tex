\doxysubsection{Detailed Description}
Інтерфейс для алгоритмів сортування. 

Цей інтерфейс задає контракт для будь-\/якого класу, що реалізує алгоритм сортування. Дає змогу працювати з різними реалізаціями (послідовними, паралельними тощо) за єдиним інтерфейсом.
