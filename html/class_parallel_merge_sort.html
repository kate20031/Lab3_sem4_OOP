Detailed Description

Клас, що реалізує паралельний алгоритм сортування злиттям.

Виконує сортування з використанням мультипоточності, розбиваючи задачу на підзадачі, які виконуються паралельно до досягнення максимальної глибини розбиття. Наслідує інтерфейс ISorter.
